# Creator System

## 1. Purpose

The Creator System defines:

- how users become creators  
- how games are submitted and approved  
- how creators earn rewards  

This system ensures:

> Creators can contribute value while remaining constrained, safe, and aligned with platform rules.

---

## 2. Core Principles

---

### 2.1 Constrained Creation

- creators cannot:
  - build arbitrary systems  
- must use:
  - Game SDK templates  

---

---

### 2.2 Platform-Controlled Monetization

- creators do not:
  - directly control monetization  
- revenue is:
  - system-distributed  

---

---

### 2.3 Quality Over Quantity

- games must:
  - meet validation thresholds  

---

---

### 2.4 Anti-Exploit by Design

- creators cannot:
  - design games that farm rewards  

---

---
