# CORE DOCTRINE

## Purpose
Define the non-negotiable principles that govern all EndlessGames systems.

## Principles

### 1. System-Enforced Behavior > Human Moderation
All platform rules must be enforced through system design, not human review.

### 2. Player Freedom Within Hard Constraints
Users are free to act within clearly defined system boundaries.
No soft rules. Only enforceable ones.

### 3. No Hidden Systems
All major systems (economy, visibility, rewards) must be explainable and predictable.

### 4. Creator = First-Class Citizen
Creators are not users—they are system participants with elevated capabilities.

### 5. Cross-System Consistency
All systems must:
- Share identity model
- Share permission model
- Share event system

### 6. Safety Through Design
Safety is achieved by:
- Limiting actions
- Structuring interactions
- Preventing harmful states

NOT by:
- Reactive moderation
- Subjective enforcement

---

## Enforcement Rules

All systems MUST:
- Declare inputs and outputs
- Define allowed actions
- Reject invalid states automatically

If a rule cannot be enforced programmatically → it is invalid.
