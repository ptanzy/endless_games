# CONTENT_LIFECYCLE

---

## 🎯 Purpose
Define the lifecycle of all content types across EndlessGames, including media, creator content, SDK experiences, and seasonal content.

---

# 1. Phases
1. **Creation** — Content is authored or recorded  
2. **Draft** — Content is saved but not submitted  
3. **Submission** — Creator submits content for validation  
4. **Validation** — Automated and human checks  
5. **Publish** — Content becomes discoverable  
6. **Discovery** — Content appears in Hub and feeds  
7. **Engagement** — Players interact with content  
8. **Retirement** — Content is archived or deprecated  

---

# 2. Actors
- Creator  
- System Validator  
- Platform  
- Player  

---

# 3. Key Rules
- All content must pass validation  
- All content must be versioned  
- Deprecated content must be removable  
- No unsafe content may be published  
- All content must follow preset‑safe communication rules  

---

# 4. Content Categories
- Media (Tube, TV, Stream, Moments)  
- Creator content  
- SDK interactive content  
- SDK games  
- SDK modes  
- Seasonal content  
- Brand content  

---

# 5. Validation Requirements
- Safety validation  
- Asset validation  
- Metadata validation  
- Gameplay validation (SDK)  
- Seasonal compliance (if applicable)
