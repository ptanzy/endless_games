# Media Planning README

---

## What Belongs Here
- Documents for media systems, radio, TV, recaps, moments, streams, and media system overview
- Planning for media creation, publishing, and integration

## What Doesn’t Belong Here
- Core, brand, cosmetic, creator, economy, social, or tools-specific planning
- Implementation details or technical specs

## Relationship to Other Folders
- Media may feature brand, creator, and social content
- Media safety and publishing are informed by core doctrines and tools
- Media rewards and engagement may tie into economy and events
