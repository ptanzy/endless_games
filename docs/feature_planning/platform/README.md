# Platform Planning README

---

## What Belongs Here
- Documents for platform expansion, integrations, analytics, performance, release, security, storage, support, systems, testing, tools, and UI
- Planning for platform-wide infrastructure, services, and technical operations

## What Doesn’t Belong Here
- Core, brand, cosmetic, creator, economy, media, or social-specific planning
- Implementation details or game-specific docs

## Relationship to Other Folders
- Platform services support all planning pillars
- Platform security and performance are governed by core doctrines and standards
- Platform integrations may tie into brand, creator, economy, and media systems
