# MODERATION_STRATEGY

---

## 🎯 Purpose
Define the minimal human‑in‑the‑loop moderation strategy for EndlessGames, aligned with the zero‑moderation doctrine.

---

# 1. Principles
- AI‑first, human‑verified  
- Preset‑only communication  
- No open text  
- No unsafe content  
- No personal data  

---

# 2. Moderation Scope
- Creator content  
- SDK experiences  
- Media uploads  
- Brand content  
- Seasonal content  

---

# 3. Human‑In‑The‑Loop
Humans review:
- Edge cases  
- AI‑flagged content  
- Brand integrations  
- High‑visibility content  

---

# 4. Rules
- AI cannot make irreversible decisions  
- All moderation actions must be logged  
- All decisions must be explainable  

---

# 5. Anti‑Patterns
- Human‑only moderation  
- AI‑only moderation  
- Unreviewed brand content  

---
