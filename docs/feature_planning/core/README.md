# Core Planning README

---

## What Belongs Here
- Documents defining foundational platform rules, doctrines, and systems
- Planning for core gameplay, progression, achievements, account structure, platform expansion, and performance
- Security standards and platform-wide constraints

## What Doesn’t Belong Here
- Brand, cosmetic, creator, economy, media, social, or tools-specific planning
- Implementation details or technical specs (should be in architecture or platform folders)

## Relationship to Other Folders
- Core rules and doctrines govern all other planning pillars
- Security standards inform asset validation in tools and content safety in media/social
- Account structure and progression systems integrate with economy, social, and creator docs
