# Cosmetic Types Planning

This document details the various cosmetic types available on the platform, their rarity, acquisition methods, and customization options.

---

## 🎯 Purpose
Define the full taxonomy of cosmetic types supported by EndlessGames, ensuring consistency, clarity, and cross‑game compatibility across all cosmetic surfaces.

---

# 1. Goals
- Provide a unified cosmetic type system  
- Ensure all cosmetics are preset‑based and safe  
- Support expressive identity across all games  
- Enable rarity, prestige, and seasonal variants  

---

# 2. Personas
- **Player:** Wants expressive cosmetic options  
- **Collector:** Wants variety and rarity  
- **Platform:** Needs a consistent taxonomy  

---

# 3. User Stories
- *As a player, I want different cosmetic types to express myself.*  
- *As a collector, I want rare versions of cosmetic types.*  
- *As the platform, I want a clean, scalable cosmetic taxonomy.*  

---

# 4. Core Capabilities
- Cosmetic type definitions  
- Rarity variants  
- Seasonal variants  
- Prestige variants  
- Cross‑game compatibility rules  

---

# 5. Cosmetic Types

### **5.1 Avatars**
- Player identity images  
- Seasonal, prestige, and event variants  

### **5.2 Frames**
- Decorative borders for profiles and leaderboards  

### **5.3 Titles**
- Text‑based identity badges (preset‑only, no custom text)  

### **5.4 Badges**
- Achievement‑based or event‑based icons  

### **5.5 Emotes**
- Predefined animations or reactions  

### **5.6 Effects**
- Visual flourishes (sparkles, glows, trails)  

### **5.7 Animations**
- Victory poses, idle animations, intros  

### **5.8 Game‑Specific Skins**
- Chess pieces  
- Board themes  
- Card backs  
- UI themes  

---

# 6. Rules & Constraints
- No custom text  
- No UGC uploads  
- No unsafe imagery  
- All cosmetics must be preset‑based  
- All cosmetics must be cross‑game compatible unless flagged  

---

# 7. Deep Dive

### **7.1 Rarity Variants**
Each cosmetic type supports:
- Common  
- Rare  
- Epic  
- Legendary  
- Prestige  

### **7.2 Seasonal Variants**
- Winter  
- Summer  
- Halloween  
- Lunar New Year  
- Anniversary  

### **7.3 Prestige Variants**
- Earned through long‑term progression  
- Never purchasable  
- Always rare and exclusive  

---

# 8. Success Criteria
- Clear cosmetic taxonomy  
- Easy for players to understand  
- Easy for platform to scale  
- No moderation required
