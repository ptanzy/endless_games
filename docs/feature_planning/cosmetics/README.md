# Cosmetics Planning README

---

## What Belongs Here
- Documents for cosmetic system design, cosmetic types, bundles, pipelines, and prestige cosmetics
- Planning for cosmetic creation, validation, and integration

## What Doesn’t Belong Here
- Brand, core, creator, economy, media, social, or tools-specific planning
- Implementation details or technical specs

## Relationship to Other Folders
- Cosmetics may originate from brand, creator, or internal tools
- Cosmetic validation is informed by core security standards and tools
- Cosmetic rewards may be tied to economy and progression systems
