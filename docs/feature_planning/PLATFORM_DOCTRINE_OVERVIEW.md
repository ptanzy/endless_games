# PLATFORM_DOCTRINE_OVERVIEW

---

## 🎯 Purpose
Define the non‑negotiable principles that every EndlessGames system must follow.

---

## 🧱 Core Platform Principles

### Zero Moderation
- No open text, no voice, no webcam
- No external audio or user‑uploaded media
- All communication is preset‑only
- All media is gameplay‑only

### Player‑First Monetization
- No pay‑to‑win
- No loot boxes
- No forced ads
- Ads, offers, and memberships are optional
- All monetization is cosmetic‑only or progression‑adjacent, never power

### Unified Identity
- One profile across all games and media
- Shared cosmetics, ranks, achievements, clans
- Identity is gameplay‑centric, not personal‑data‑centric

### Creator‑Powered
- Safe creator economy
- Gameplay‑only media surfaces
- Brand deals and sponsorships are platform‑mediated

---

## 🔐 Global Safety Rules

- No user‑generated audio/video
- No arbitrary text input
- No external uploads into player‑facing surfaces
- All assets and presets must be reviewed and approved
- Child‑safe by default; no mature themes

---

## 🎨 Cosmetic Doctrine

- Cosmetics are visual only
- No gameplay advantage, no stats, no power
- Brand swag, licensed items, prestige items all obey cosmetic‑only rules

---

## 📏 Doctrine Enforcement

Every planning doc must:
- Explicitly avoid features that require human moderation
- Respect preset‑only and gameplay‑only constraints
- Avoid mechanics that create power imbalance
- Defer to this document when in doubt
