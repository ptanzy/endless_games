# TOOLS_OVERVIEW

---

## 🎯 Purpose
Define the internal and creator‑facing tools that support EndlessGames operations, creation, and governance.

---

# 1. Tool Categories

### Creator Tools
- Media uploader  
- Interactive content builder  
- Game SDK editor  
- Mode SDK editor  
- Creator analytics  

### Platform Tools
- Content review dashboard  
- SDK validation console  
- Preset management tool  
- Seasonal configuration tool  
- Brand integration console  

### Operational Tools
- Monitoring dashboards  
- Alerting systems  
- Deployment tools  
- Load testing tools  

---

# 2. Rules
- Tools must enforce permissions  
- Tools must be auditable  
- Tools must follow safety doctrine  
- Tools must not expose internal system details  

---

# 3. Anti‑Patterns
- Tools bypassing validation  
- Tools granting implicit permissions  
- Tools modifying authoritative data directly
