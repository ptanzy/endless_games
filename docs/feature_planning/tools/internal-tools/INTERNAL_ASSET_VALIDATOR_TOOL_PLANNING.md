# INTERNAL_ASSET_VALIDATOR_TOOL_PLANNING

---

## 🎯 Purpose

The Internal Asset Validator ensures all assets meet EndlessGames safety, performance, and compliance requirements before publishing.

---

# 1. Goals

- Automate asset validation  
- Prevent unsafe or non‑compliant assets  
- Reduce manual review workload  
- Provide actionable feedback  

---

# 2. Core Capabilities

## 2.1 File Validation
- File format  
- File size  
- Texture resolution  
- Poly count  

## 2.2 Safety Validation
- No hidden metadata  
- No inappropriate shapes  
- No text  
- No symbols  
- Child‑safe palettes  

## 2.3 Performance Validation
- LOD checks  
- Rig compatibility  
- Animation compatibility  

## 2.4 Reporting
- Pass/fail  
- Detailed error list  
- Suggested fixes  

---

# 3. Workflow

1. Upload asset  
2. Run validation  
3. Review results  
4. Fix issues  
5. Re‑validate  

---

# 4. Success Criteria

- Zero unsafe assets reach staging  
- Validation is fast and reliable  
- Teams can self‑serve asset checks
