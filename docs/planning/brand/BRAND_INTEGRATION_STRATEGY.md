# BRAND_INTEGRATION_STRATEGY

---

## 🎯 Purpose
Define how brands integrate into EndlessGames in a safe, cosmetic‑only, preset‑only manner.

---

# 1. Principles
- Brand integrations must be safe  
- No gameplay advantages  
- Cosmetic‑only  
- Preset‑only communication  
- Fully reviewable  
- Time‑bound  

---

# 2. Integration Types
- Brand cosmetics  
- Brand Hub modules  
- Brand creator partnerships  
- Brand seasonal arcs  
- Brand SDK experiences (template‑restricted)  

---

# 3. Rules
- No open text  
- No external assets without review  
- No brand‑driven gameplay changes  
- All brand content must pass validation  

---

# 4. Brand Workflow
1. Brand proposal  
2. Template selection  
3. Asset submission  
4. Validation  
5. Integration  
6. Launch  
7. Retirement  

---

# 5. Anti‑Patterns
- Unbounded brand presence  
- Unsafe messaging  
- Gameplay‑affecting brand items  

---
