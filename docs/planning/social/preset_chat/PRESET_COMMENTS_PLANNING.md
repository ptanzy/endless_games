# PRESET_COMMENTS_PLANNING

---

## 🎯 Purpose
Define the preset comment system used across EndlessTube, EndlessStream, profiles, and clan pages to enable safe, structured communication.

---

# 1. Goals
- Provide structured, expressive comments  
- Maintain zero‑moderation safety  
- Support creators and social identity  
- Enable cosmetic comment packs (future)  

---

# 2. Personas
- **Player:** Wants to comment safely  
- **Creator:** Wants engagement  
- **Platform:** Needs zero moderation  

---

# 3. User Stories
- *As a player, I want to leave safe comments on videos.*  
- *As a creator, I want preset comments that boost engagement.*  
- *As the platform, I want comments that require no moderation.*  

---

# 4. Core Capabilities
- Preset comment library  
- Context‑aware comment categories  
- Creator‑specific comment sets  
- Seasonal comment packs  

---

# 5. Workflows

### **5.1 Comment Workflow**
1. Player selects comment category  
2. Player selects preset  
3. Comment is posted  
4. No unsafe content possible  

---

# 6. Rules & Constraints
- No open text  
- No emojis outside preset packs  
- No personal data exchange  
- All comments must be reviewed  

---

# 7. Deep Dive

### **7.1 Comment Categories**
- Praise  
- Humor  
- Support  
- Hype  
- GG / Well Played  
- Creator‑focused comments  

### **7.2 Monetization**
- Cosmetic comment packs (future)  
- Seasonal comment sets  

---

# 8. Success Criteria
- High comment usage  
- Strong creator engagement  
- Zero moderation incidents  
- Comments feel expressive despite presets
