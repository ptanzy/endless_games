# EndlessOS Security — Overview

## Purpose
This folder defines all security best practices for EndlessOS across three pillars:

- UI Security  
- Backend Security  
- Data Security  

These documents ensure the platform remains deterministic, safe, and resistant to attacks.

---

# Structure

## UI Security
Focuses on:
- XSS prevention  
- CSRF prevention  
- safe rendering  
- iframe safety  
- third‑party content isolation  

## Backend Security
Focuses on:
- service‑to‑service trust  
- API security  
- authentication & sessions  
- backend validation  
- rate limiting  
- secrets  
- logging  
- configuration security  

## Data Security
Focuses on:
- encryption  
- access control  
- classification  
- retention  
- deletion  
- backups  
- integrity  
- governance  

---

# Usage
These documents are authoritative.  
All engineering work must comply with the rules defined here.
