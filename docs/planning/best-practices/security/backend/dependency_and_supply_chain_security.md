# EndlessOS — Dependency & Supply Chain Security

## Purpose
Defines how dependencies must be managed to prevent supply‑chain attacks.

---

# Core Principles

## 1. Pin Versions
- exact versions  
- no floating versions  

---

## 2. Audit Dependencies
- vulnerability scans  
- remove unused packages  
- avoid untrusted sources  

---

## 3. Minimize Dependency Chains
- prefer native code  
- avoid unnecessary libraries  

---

# Summary
Supply chain security protects EndlessOS from compromised dependencies.
