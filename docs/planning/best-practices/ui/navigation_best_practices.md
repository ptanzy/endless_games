# navigation best practices

## Overview
Design intuitive navigation to help users find content and features easily.

## Best Practices
- Use clear, descriptive labels for navigation items.
- Keep navigation structure shallow and logical.
- Highlight the current location within the app.
- Provide easy access to primary actions.
- Support keyboard and screen reader navigation.
- Ensure navigation is responsive and touch-friendly.
- Avoid hidden or ambiguous navigation paths.
