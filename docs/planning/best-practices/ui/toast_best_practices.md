# toast best practices

## Overview
Use toast notifications for brief, non-intrusive feedback.

## Best Practices
- Display toasts in a consistent location (e.g., bottom right).
- Keep messages short and actionable.
- Auto-dismiss toasts after a reasonable time.
- Allow users to manually dismiss toasts.
- Avoid using toasts for critical errors or confirmations.
- Ensure toasts are accessible and screen reader friendly.
- Limit the number of simultaneous toasts.
