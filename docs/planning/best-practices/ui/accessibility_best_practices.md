# accessibility best practices

## Overview
Design and build accessible interfaces for all users, including those with disabilities.

## Best Practices
- Use semantic HTML and ARIA roles appropriately.
- Ensure sufficient color contrast for text and UI elements.
- Support keyboard navigation throughout the UI.
- Provide text alternatives for non-text content.
- Test with screen readers and assistive technologies.
- Avoid relying solely on color to convey information.
- Regularly audit and improve accessibility.
