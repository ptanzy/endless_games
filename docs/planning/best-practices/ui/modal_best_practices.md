# modal best practices

## Overview
Use modals to focus user attention without disrupting workflow.

## Best Practices
- Use modals sparingly for critical tasks or information.
- Provide clear actions (e.g., confirm, cancel) in modals.
- Trap focus within the modal and restore on close.
- Allow modals to be dismissed with ESC or outside click.
- Ensure modals are accessible to screen readers.
- Avoid stacking multiple modals.
- Animate modal transitions for context.
