# popover best practices

## Overview
Use popovers for lightweight, contextual information or actions.

## Best Practices
- Trigger popovers with clear, intentional actions.
- Position popovers relative to their trigger.
- Dismiss popovers on outside click or ESC.
- Keep popover content concise and focused.
- Ensure popovers are accessible and keyboard-navigable.
- Avoid using popovers for critical or persistent information.
- Test popover behavior on all devices.
