# SDK_ASSET_REQUIREMENTS

---

## 🎯 Purpose
Define the rules and constraints for all assets used in SDK projects.

---

# 1. Asset Types
- Textures  
- Models  
- Audio  
- Animations  
- UI elements  
- Scripts  

---

# 2. Requirements

### Size Limits
- Max file size  
- Max total project size  

### Format Requirements
- Approved formats only  
- Compression required  
- Naming conventions  

### Safety Requirements
- No unsafe imagery  
- No copyrighted content  
- No external URLs  

### Performance Requirements
- Texture resolution limits  
- Poly count limits  
- Audio bitrate limits  

---

# 3. Asset Inspector
- Preview assets  
- Validate metadata  
- Check performance  
- Check safety  

---

# 4. Anti‑Patterns
- Raw uncompressed assets  
- External asset references  
- Unsafe or unreviewed content  

---
