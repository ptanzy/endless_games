# SDK_EDITOR_WORKFLOW

---

## 🎯 Purpose
Define the workflow creators follow when building SDK content using the EndlessCreator SDK Editor.

---

# 1. Workflow Steps

### 1. Create Project
- Select template  
- Configure metadata  
- Initialize versioning  

### 2. Build Content
- Add assets  
- Configure scenes  
- Define logic  
- Set up interactions  
- Configure UI  

### 3. Validate
- Run validation pipeline  
- Fix errors  
- Re‑validate  

### 4. Preview
- Local preview  
- Simulated multiplayer  
- Performance profiling  

### 5. Package
- Build package  
- Inspect output  
- Generate version  

### 6. Publish
- Submit for review  
- Track review status  
- Publish to Hub  

---

# 2. Editor Features
- Scene editor  
- Logic graph editor  
- Asset inspector  
- Template manager  
- Preview simulator  
- Debug console  

---

# 3. Rules
- Editor must enforce safety  
- Editor must enforce asset limits  
- Editor must prevent unsafe logic  
- Editor must integrate with validation pipeline  

---

# 4. Anti‑Patterns
- Editing without validation  
- Manual file manipulation  
- Bypassing the editor
