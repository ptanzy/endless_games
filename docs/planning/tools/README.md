## What Belongs Here
- Documents for internal tools, dev tools, asset validators, dashboards, and content publishing pipelines

- Core, brand, cosmetic, creator, economy, media, or social-specific planning
- Implementation details or technical specs

- Tools support content creation, validation, and publishing for all pillars
- Tools enforce core doctrines and security standards
- Tools workflows may integrate with brand, creator, and media systems
