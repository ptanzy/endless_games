# PLAYER_LIFECYCLE

---

## 🎯 Purpose
Define the end-to-end lifecycle of a player within EndlessGames, from acquisition to reactivation.

---

# 1. Stages

### 1. Acquisition
- Marketing  
- Social discovery  
- Creator-driven discovery  

### 2. Onboarding
- Identity setup  
- Hub introduction  
- Preset communication introduction  
- Starter cosmetics  

### 3. Engagement
- Gameplay  
- Media consumption  
- Creator interaction  
- Seasonal participation  

### 4. Retention
- Progression  
- Quests  
- Events  
- Pass rewards  

### 5. Churn
- Inactivity  
- Session drop-off  

### 6. Reactivation
- Seasonal resets  
- Creator notifications  
- Hub personalization  

---

# 2. Systems Involved
- Identity System  
- Progression System  
- Event System  
- Economy System  
- Analytics Pipeline  
- EndlessHub  

---

# 3. Key Rules
- Onboarding must be frictionless  
- Engagement must be safe and preset-only  
- Retention must be reward-driven, not punitive  
- Reactivation must be personalized
