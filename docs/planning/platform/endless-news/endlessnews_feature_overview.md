# EndlessNews — Platform Overview & Feature Planning Document

## 1. Purpose
EndlessNews is a low‑effort, high‑value news aggregation platform designed to give users:

- clean, trustworthy summaries  
- contextual understanding  
- a sense of global patterns  
- a weekly human editorial voice  
- a deterministic, cinematic EndlessOS experience  

It is not a newsroom.  
It is a smart viewer for the world.

---

## 2. Core Concept
EndlessNews combines:

- automated aggregation  
- structural trust scoring  
- contextual world‑event mapping  
- visual storytelling  
- optional weekly editorials  

…into a single, unified platform.

The goal is to make news understandable, transparent, and context‑rich without requiring a large editorial team.

---

## 3. Core Features (MVP)

### 3.1 News Feed
A deterministic Endless‑style feed showing:
- headline  
- summary  
- source  
- timestamp  
- category  
- thumbnail  

### 3.2 Category Tabs
- World  
- Tech  
- Gaming  
- Entertainment  
- Sports  
- Finance  

### 3.3 Article Detail View
- headline  
- summary  
- source  
- trust meter  
- context panel  
- “Open Source Article” button  

### 3.4 Trust Meter
A structural, non‑political scoring system based on:
- source reputation  
- emotional language  
- factual density  
- cross‑source agreement  
- speculative/adversarial framing  

### 3.5 Context Panel
Shows:
- related stories  
- timeline  
- key actors  
- regional context  
- similar past events

## 7. Backend Architecture (Lightweight)

### 7.1 Aggregation Layer
Pulls from:
- NewsAPI  
- Mediastack  
- GNews  
- Reddit RSS  
- Hacker News  
- YouTube trending  

### 7.2 Ranking Layer
Scores stories based on:
- recency  
- trust meter  
- relevance  
- cross‑source agreement  

### 7.3 Cache Layer
Stores:
- top stories  
- category feeds  
- trust scores  
- context graphs  

### 7.4 API Layer
Endpoints:
- `/news`  
- `/news/:id`  
- `/news/categories`  
- `/news/context/:id`  
- `/news/trust/:id`

## 10. Vision Statement
EndlessNews is not trying to compete with newsrooms.  
It’s trying to make the world legible.

It’s a platform where:
- news is understandable  
- context is automatic  
- trust is transparent  
- bias is structural, not political  
- the user feels informed, not overwhelmed  

This is the EndlessOS way.
