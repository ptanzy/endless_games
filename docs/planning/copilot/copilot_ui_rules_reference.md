# Copilot UI Rules Reference

## Purpose
A distilled reference of EndlessOS UI doctrine for Copilot to apply automatically.

---

# Core UI Rules

## 1. Neutral Palette Only
- No platform colors in UI components  
- No accent colors in buttons, icons, or surfaces  

## 2. Deterministic Layout
- No adaptive resizing  
- No dynamic layout shifts  
- No content‑dependent spacing  

## 3. Motion Rules
- Fade only  
- 2–4px micro‑shift  
- 120–150ms duration  
- No slide, bounce, parallax, scale, or rotation  

## 4. Safe Zones
- All UI must remain inside safe zones  
- No clipping  
- No edge collisions  

## 5. Typography
- Use platform typography rules  
- No custom fonts in system UI  

---

# Summary
Copilot must enforce these UI rules in all generated components and layouts.
