# Copilot Prohibited Patterns

## Purpose
Defines patterns Copilot must never generate.

---

# Prohibited UI Patterns
- Slide animations  
- Bounce animations  
- Parallax  
- Scaling transitions  
- Adaptive layout  
- Dynamic spacing  
- Platform‑colored UI  
- Context‑dependent behavior  

---

# Prohibited Navigation Patterns
- Animated scroll  
- Sliding page transitions  
- Non‑deterministic routing  
- Auto‑redirects  

---

# Prohibited Component Patterns
- Stateful components without need  
- Deeply nested components  
- Layout‑shifting components  

---

# Summary
Copilot must avoid all prohibited patterns to maintain EndlessOS consistency.
