# WELLBEING_MODE_PLANNING

---

## 🎯 Purpose & Philosophy

Wellbeing Mode is a **platform‑wide comfort, accessibility, and healthy‑habit system** designed to reduce overstimulation, support sustainable play, and create a calm, predictable experience for players of all ages.

The philosophy is built on:

- **Player autonomy** — wellbeing features are optional and empowering  
- **Comfort-first design** — reduced motion, softer visuals, calmer pacing  
- **Healthy digital habits** — gentle, non‑judgmental time awareness  
- **Child safety** — Wellbeing Mode ON by default for under‑13 accounts  
- **Zero Moderation alignment** — no communication vectors, no unsafe content  
- **Games‑only child experience** — no media surfaces to manage  
- **Consistent UX** — predictable behavior across all games and surfaces  

Wellbeing Mode is not restrictive — it is **supportive, calming, and player‑first**.

---

## 🧩 Problems This Solves

- Some players are sensitive to motion, flashing, or high‑contrast visuals  
- Long sessions can cause fatigue without gentle awareness tools  
- Media surfaces previously added overstimulation (now removed for children)  
- Children require calmer, developmentally appropriate UI  
- Parents want tools that support healthy screen habits  
- EndlessGames needs a wellbeing layer that scales across expansions  

Wellbeing Mode solves these by providing **optional, platform‑wide comfort and time‑awareness tools**.

## 📐 High‑Level Description

Wellbeing Mode is a **toggleable platform mode** that adjusts:

- Visual intensity  
- Motion levels  
- Color contrast  
- Notification style  
- Time‑awareness surfaces  

It integrates with:

- Session Time Insights  
- Parental Account System  
- Game UI  
- EndlessHub (adult accounts only)  
- Platform Expansion rules  

Because child accounts are **games‑only**, Wellbeing Mode is simpler and more predictable for minors.

---

## 🧵 Core System Components

### 1. Reduced Motion Layer
- Disables non‑essential animations  
- Softens transitions  
- Reduces parallax and camera motion  
- Slows UI movement  
- Removes rapid visual effects  

### 2. Reduced Stimulus Visual Theme
- Softer color palette  
- Lower contrast (optional)  
- Calmer highlight states  
- Reduced flashing  
- No rapid pulsing or strobing  

### 3. Gentle Notification System
- Soft fades instead of pops  
- No vibration (optional)  
- No high‑contrast alerts  
- No urgency language  
- Optional “quiet mode”  

### 4. Time‑Awareness Integration
- Session Time Insights displayed softly  
- Optional break reminders  
- Optional session caps  
- Optional bedtime cutoffs (child accounts)  

### 5. Games‑Only Child Integration
Because child accounts have **no media surfaces**, Wellbeing Mode for minors focuses on:

- Reduced‑stimulus UI  
- Time limits  
- Bedtime cutoffs  
- Break reminders  
- Calm, predictable game transitions  

### 6. Adult Media Surface Adjustments
For adult accounts only:

- No autoplay  
- No infinite scroll  
- No binge playlists  
- Softer overlays  
- Reduced‑stimulus transitions

## 🔄 Core Flows

### 1. Wellbeing Mode Activation Flow
Player → Settings → Enable Wellbeing Mode → UI Adjusts Instantly

### 2. Game Flow
Wellbeing Mode On → Reduced Motion → Softer Visuals → Time‑Awareness Active

### 3. Time‑Awareness Flow
Wellbeing Mode On → Optional Reminders → Soft Notifications → No Interruptions

### 4. Child Account Flow
Child Account → Wellbeing Mode Enabled by Default → Parent Can Lock It

## 📏 Rules & Constraints

### Safety
- No communication vectors  
- No external content  
- No unsafe media  

### Visual Comfort
- No flashing  
- No strobing  
- No rapid transitions  
- No autoplay (adult media only)  
- No infinite scroll (adult media only)  

### Time‑Awareness
- Optional  
- Non‑judgmental  
- Non‑interruptive  
- Integrated with Session Time Insights  

### Child Accounts
- Wellbeing Mode enabled by default  
- Parents can lock it  
- No media surfaces to adjust  
- Bedtime cutoffs and time limits enforced  

### Zero Moderation Alignment
- No text input  
- No voice  
- No uploads  
- No user‑generated content  

### Platform Expansion Alignment
- All new games must support Wellbeing Mode  
- All new surfaces must include reduced‑stimulus variants (adult only)  

---

## 🔗 Interactions With Other Systems

### Session Time Insights
- Soft display of session duration  
- Optional reminders  
- Optional bedtime cutoffs  

### Parental Account System
- Wellbeing Mode defaults to ON for child accounts  
- Parents can lock or unlock it  

### Media Systems (Adult Only)
- Tube, Stream, Radio, TV adjust to reduced‑stimulus mode  
- Child accounts have no access  

### Zero Moderation
- No new communication vectors introduced  

### Platform Expansion
- All new games must support reduced motion and soft notifications  

### Cosmetic System
- Reduced motion affects cosmetic animations  
- Optional “calm mode” for cosmetics  

---

## ⚠️ Risks & Considerations

- Over‑reduction may make UI feel slow for some players  
- Must avoid making wellbeing feel punitive  
- Must avoid guilt‑inducing language  
- Must ensure accessibility and wellbeing do not conflict  
- Must ensure game feel remains responsive
