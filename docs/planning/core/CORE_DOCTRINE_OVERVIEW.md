# CORE_DOCTRINE_OVERVIEW

---

## 🎯 Purpose & Philosophy

The Core Doctrine defines the **non‑negotiable platform rules** that govern every system within EndlessGames. These doctrines ensure:

- Zero moderation  
- Player safety  
- Child safety  
- Wellbeing  
- Predictable expansion  
- Deterministic backend behavior  
- Consistent UX  
- Brand safety  
- Parent trust  

Every game, system, and future expansion must obey these doctrines without exception.

**Child accounts are now strictly *games‑only*.  
No media surfaces.  
No social profiles.  
No public visibility.**

This dramatically simplifies safety, compliance, and long‑term platform evolution.

---

# 1. ZERO MODERATION DOCTRINE

Zero Moderation eliminates the need for human moderation by removing all vectors of unsafe communication or user‑generated content.

## 1.1 Communication Rules
- No open text  
- No voice chat  
- No webcam  
- No external audio  
- No external video  
- No file uploads  
- No user‑generated content  
- No custom emojis  
- No custom overlays  
- Preset‑only communication everywhere  

## 1.2 Media Rules
- Gameplay‑only media for adult accounts  
- No commentary  
- No facecam  
- No external editing  
- No external music  
- No user‑uploaded images or videos  

## 1.3 Platform Rules
- No moderation team  
- No content review pipeline  
- No unsafe content vectors  
- All systems must be safe by design  

## 1.4 Inheritance Rules
- Child accounts inherit Zero Moderation  
- All expansions must pass Zero Moderation review  
- All new media surfaces must follow gameplay‑only rules  

---

# 2. CHILD SAFETY DOCTRINE

Child accounts (under 13) are **games‑only**.  
They have **no media surfaces** and **no social profiles**.

## 2.1 Account Structure
- Parent accounts include up to **4 child accounts**  
- Child accounts are **paid** and **ad‑free**  
- Child accounts are under‑13 restricted  
- Parents can convert a child account to a regular account  

## 2.2 Game Access Rules
- Parents choose which games each child can access  
- Child accounts can NEVER access mature games  
- Per‑game time limits supported  

## 2.3 Media Access Rules (New Doctrine)
### **Child accounts have:**
- ❌ No EndlessTube  
- ❌ No EndlessStream  
- ❌ No EndlessRadio  
- ❌ No EndlessTV  
- ❌ No media surfaces of any kind  
- ❌ No publishing  
- ❌ No streaming  
- ❌ No recordings  
- ❌ No creator visibility  

### Children can only:
- Play approved games  
- Use preset reactions  
- Use preset comments  
- Earn in‑game rewards  

## 2.4 Time Management Rules
Parents can set:

- Daily time limits  
- Weekly time limits  
- Per‑game limits  
- Session caps  
- Bedtime cutoffs  
- Break reminders  

## 2.5 Child Account Restrictions
Child accounts cannot:

- Use media surfaces  
- Create content  
- Upload content  
- Stream gameplay  
- Post recordings  
- Earn payouts  
- Participate in sponsorships  
- Follow creators  
- Lead clans  
- Access brand content  
- Have public profiles  
- Be discoverable  
- Have followers

# 3. WELLBEING DOCTRINE

Wellbeing Mode provides a calm, reduced‑stimulus, supportive experience for all players — and is **enabled by default** for child accounts.

## 3.1 Visual Comfort Rules
- Reduced motion  
- Reduced stimulus  
- Softer color palette  
- No flashing  
- No strobing  
- No rapid transitions  

## 3.2 Media Comfort Rules
- No autoplay  
- No infinite scroll  
- No binge playlists  
- No overstimulating UI  

## 3.3 Notification Rules
- Soft notifications  
- No urgency language  
- Optional quiet mode  

## 3.4 Time‑Awareness Integration
- Session Time Insights displayed softly  
- Optional break reminders  
- Optional session caps  
- Optional bedtime cutoffs  

## 3.5 Child Account Integration
- Wellbeing Mode ON by default  
- Parents can lock it  
- No media surfaces to adjust  
- Time limits and bedtime cutoffs enforced

# 5. PLATFORM EXPANSION DOCTRINE

Platform Expansion defines how EndlessGames grows safely and coherently.

## 5.1 Expansion Principles
All expansions must be:

- Safe  
- Predictable  
- Scalable  
- Zero‑moderation compliant  
- Child‑safe  
- Wellbeing‑aligned  
- Deterministic  
- Consistent with platform identity  

## 5.2 Expansion Categories
- New games  
- New social systems  
- New creator tools  
- New brand systems  
- New regions  
- New platform capabilities  
- New media surfaces (adult‑only)  

## 5.3 Expansion Constraints
All expansions must integrate with:

- Zero Moderation  
- Child Safety  
- Wellbeing Mode  
- Session Time Insights  
- Parental controls  
- Deterministic backend rules  
- Unified documentation  
- Consistent UX patterns  

## 5.4 Child Safety Integration
All expansions must:

- Inherit child‑safe defaults  
- Respect parental controls  
- Respect time limits  
- Respect bedtime cutoffs  
- Block mature content for child accounts  
- Provide **no media surfaces** to child accounts
