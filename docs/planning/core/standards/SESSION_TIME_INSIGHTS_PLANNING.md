# SESSION_TIME_INSIGHTS_PLANNING

---

## 🎯 Purpose & Philosophy

Session Time Insights is a **gentle, optional, non‑judgmental time‑awareness system** designed to help players understand their play patterns without pressure, guilt, or interruption.

The philosophy is built on:

- **Player autonomy** — insights are optional and private  
- **Wellbeing-first design** — soft, ambient, non‑intrusive surfaces  
- **Child safety** — stricter defaults for under‑13 accounts  
- **Zero Moderation alignment** — no communication vectors, no unsafe content  
- **Games‑only child experience** — no media surfaces to track  
- **Developmentally responsible design** — supports healthy digital habits  

Session Time Insights is informational, not corrective.

---

## 🧩 Problems This Solves

- Players may lose track of time during long sessions  
- Parents need visibility and control for child accounts  
- EndlessGames needs a wellbeing‑aligned time system  
- Traditional “playtime warnings” are intrusive and guilt‑inducing  
- Media surfaces previously added complexity (now removed)  
- Child accounts require predictable, developmentally appropriate limits  

Session Time Insights solves these by providing **ambient, optional, player‑controlled time awareness**.

## 📐 High‑Level Description

Session Time Insights provides:

- Session duration  
- Daily playtime  
- Weekly playtime  
- Per‑game usage  
- Optional reminders  
- Optional break suggestions  
- Optional session caps  
- Optional bedtime cutoffs (child accounts)  

All insights are:

- Private  
- Local  
- Non‑judgmental  
- Non‑interruptive  
- Never shared socially  
- Never used for monetization or behavioral nudging  

This system is **supportive**, not restrictive.

---

## 🧵 Core System Components

### 1. Session Tracking Engine
Tracks:

- Current session duration  
- Daily total  
- Weekly total  
- Per‑game usage  

Tracking is:

- Local  
- Private  
- Lightweight  
- Non‑behavioral  

### 2. Insight Surfaces
Insights appear on:

- Profile dashboard  
- Optional session overlay  
- Optional end‑of‑session summary  
- Wellbeing Mode surfaces  
- Parental dashboard (child accounts)  

### 3. Optional Reminders
Players may enable:

- Soft reminders (e.g., “You’ve been playing for 1 hour”)  
- Time‑of‑day reminders  
- Break suggestions  
- Session caps  

All reminders are:

- Optional  
- Non‑blocking  
- Non‑judgmental  

### 4. Child Account Time Controls
Parents can set:

- Daily time limits  
- Weekly time limits  
- Per‑game limits  
- Session caps  
- Bedtime cutoffs  
- Break reminders  

### 5. Games‑Only Integration
Because child accounts have **no media surfaces**, the system is simpler:

- No per‑surface limits  
- No media tracking  
- No media‑specific reminders  
- No autoplay considerations  
- No binge‑prevention logic  

Only **games** are tracked.

## 🔄 Core Flows

### 1. Session Tracking Flow
Session Start → Timer Begins → Local Tracking → Optional Insights → Session End → Summary (optional)

### 2. Reminder Flow
Player Opt‑In → Reminder Settings → Soft Notification → No Interruption

### 3. Child Time Limit Flow
Parent Sets Limits → System Tracks Usage → Soft Notifications → Lockout (optional)

### 4. Bedtime Cutoff Flow
Parent Sets Cutoff → System Monitors Time → Soft Warning → Lockout (optional)

### 5. Game‑Only Flow
Player Plays Game → Time Counted Per Game → Limits Applied

## 📏 Rules & Constraints

### Safety & Privacy
- No data shared with other players  
- No social visibility  
- No public stats  
- No behavioral nudging  
- No monetization tied to time insights

- Stricter defaults  
- Bedtime cutoffs  
- Per‑game limits  
- Parent‑controlled settings  
- No media surfaces to track

- No autoplay (media removed for children)  
- No infinite scroll  
- No guilt language  
- No interruptions  

### Zero Moderation Alignment
- No text input  
- No communication vectors  
- No external media

## ⚠️ Risks & Considerations

- Over‑surfacing insights may feel intrusive  
- Must avoid guilt‑inducing language  
- Must avoid implying “healthy” or “unhealthy” playtime  
- Must avoid incentivizing longer sessions  
- Must ensure insights remain private  
- Must ensure child limits are predictable and consistent
