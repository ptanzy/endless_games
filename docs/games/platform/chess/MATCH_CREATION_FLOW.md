# Match Creation Flow

This document describes the process for creating and initializing chess matches.# MATCH CREATION FLOW

## Overview
Defines the user flow for creating a match, selecting rules, and configuring variants.

---

## Steps

### 1. Select Board Architecture
- Single Board  
- Double Board  
- Triple Board  
- Cube  
- Matrix  
- Fan-Fold  
- Torus  

### 2. Select Player Count
- 2–4 players  

### 3. Select Teams
- Free-for-all  
- Teams (N+W vs S+E)  

### 4. Select Rule Modules
- Movement rules  
- Promotion rules  
- Elimination rules  
- Turn order rules  
- Team rules  
- Advanced mechanics  
- Portal rules  
- Terrain rules  
- Endgame mutation rules  

### 5. Select Presets (Optional)
- Classic  
- Chaos  
- 3D Mode  
- Portal Mode  
- Hardcore Mode  

### 6. Confirm Settings
- Summary screen  
- Save as custom preset  

---

## Notes
Match creation must remain modular and extensible.
