# CHESS — CLAN & SOCIAL INTEGRATION

## Purpose
This document defines how Chess integrates with EndlessClans, EndlessStatus, and the platform’s identity systems.

---

## Clan Integration
Clans can:

- Earn Chess‑based clan XP  
- Participate in Chess clan events  
- Feature Chess Moments on clan pages  
- Earn clan cosmetics via Chess events  

Optional:
- Clan‑placed ads (opt‑in, EndlessGold to treasury)

---

## Social Integration
Chess supports:

- Preset chat in lobbies  
- Preset reactions in matches  
- Preset comments on Moments and Tube videos  
- Preset statuses (“Grinding Chess”, “Practicing Cube Mode”)  

---

## Identity Integration
Chess contributes to:

- Profile badges  
- Rank badges  
- Seasonal badges  
- Cosmetic loadouts  
- Profile themes
