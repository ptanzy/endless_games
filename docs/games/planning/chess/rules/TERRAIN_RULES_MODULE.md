# TERRAIN RULES MODULE

## Overview
This module defines dynamic board terrain mechanics that modify movement, safety, or accessibility.

---

## Terrain Types

### 1. Collapsing Squares
- Squares disappear after being stepped on.
- Pieces on collapsing squares die.

### 2. Rising Platforms
- Squares elevate or descend, affecting vertical movement.

### 3. Rotating Sections
- Sections of the board rotate 90° or 180° every X turns.

### 4. Dangerous Tiles
- Entering these tiles kills the piece.
- Can be static or dynamic.

---

## Terrain Behavior Options

### 1. Predictable Terrain
- Terrain changes follow a fixed schedule.

### 2. Random Terrain
- Terrain changes occur randomly.

### 3. Player-Triggered Terrain
- Certain moves or captures trigger terrain changes.

---

## Notes
Terrain rules are optional and intended for variant or chaotic play.
