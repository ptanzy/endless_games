# TEAM RULES MODULE

## Overview
This module defines all team-related rules for 2v2 matches.

---

## Team Structure
- Default teams: North + West vs South + East
- Custom team assignments may be added in future updates.

---

## Team Interaction Rules

### 1. No Friendly Check
- Teammates cannot check or checkmate each other.

### 2. No Pass-Through
- Teammates cannot move through each other’s pieces.

### 3. Shared Victory
- If one teammate wins, both win.

### 4. Limited Communication
- Only preset chat wheel options allowed.
- Prevents unfair coordination.

---

## Optional Team Rules

### 1. Shared Vision (Variant)
- Teammates see each other’s hidden information (for fog-of-war modes).

### 2. Shared Reserve (Variant)
- Captured pieces go into a shared pool.

---

## Notes
Team rules interact with:
- Elimination rules  
- Promotion rules  
- Turn order
