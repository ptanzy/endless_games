# PORTAL RULES MODULE

## Overview
This module defines teleportation mechanics that can be added to any board architecture.

---

## Portal Types

### 1. Paired Portals
- Two squares are linked.
- Entering one teleports the piece to the other.

### 2. Random Portals
- Teleport destination is chosen randomly from a list.

### 3. Shifting Portals
- Portal locations change every X turns.

### 4. Open/Close Cycles
- Portals alternate between active and inactive states.

---

## Portal Behavior Options

### 1. Teleport on Entry
- Teleportation occurs immediately when stepping onto a portal.

### 2. Teleport on Exit
- Teleportation occurs when leaving the portal square.

### 3. Capture Through Portal
- Pieces may capture through portals.

### 4. No Capture Through Portal
- Teleportation only; no remote captures.

---

## Notes
Portal rules interact heavily with:
- Movement rules  
- Terrain rules  
- Endgame mutation rules
