# TURN ORDER RULES MODULE

## Overview
This module defines how turn order is determined in matches with 3–4 players.

---

## Turn Order Options

### 1. Clockwise
- Default order: North → East → South → West

### 2. Counter-Clockwise
- Reverse order: North → West → South → East

### 3. Random Each Round
- At the start of each full cycle, turn order is randomized.
- Prevents predictable targeting.

### 4. Host-Selected Order
- Host manually sets the turn sequence before the match.

---

## Round Structure
- A “round” is defined as all players taking one turn.
- Randomization (if enabled) occurs only between rounds.

---

## Notes
Turn order affects:
- Checkmate timing  
- Elimination timing  
- Team coordination  
- Endgame pacing
