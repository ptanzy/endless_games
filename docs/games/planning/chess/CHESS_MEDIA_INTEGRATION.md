# CHESS — MEDIA INTEGRATION

## Purpose
This document defines how Chess integrates with EndlessTube, EndlessTV, EndlessMoments, and EndlessRecaps.

---

## EndlessTube
Chess supports:

- Automatic gameplay recording  
- Manual clip saving  
- Preset‑only comments  
- Creator monetization via impressions  

---

## EndlessTV
Chess contributes:

- Highlights  
- Checkmates  
- Promotions  
- Multi‑captures  
- 3D transitions (Cube/Matrix)  

---

## EndlessMoments
Chess triggers Moments for:

- Checkmates  
- Promotions  
- Tactical combos  
- Multi‑layer transitions  
- Portal plays  
- Terrain interactions  

---

## EndlessRecaps
Chess contributes:

- Wins  
- Rank progress  
- Achievements  
- Best Moments  
- Seasonal stats
