# TRIPLE BOARD MODE — CONCEPT DOCUMENT

## Overview
Triple Board Mode expands the stacked-board concept to three layers: Top, Middle, and Bottom. This mode is conceptual and intended for advanced play.

---

## Board Architecture
- **Top Board**
- **Middle Board**
- **Bottom Board**

### Variants
- Straight stack  
- Fan-fold / accordion topology  
- Offset layers  

---

## Movement Rules
- Same as Double Board Mode  
- Middle board acts as a contested zone  
- Access rules not yet defined  

---

## Starting Positions
- Players start on Top and Bottom boards  
- Middle board begins empty  

---

## Promotion Rules
- Not yet finalized  
- Options include:
  - Classic  
  - Middle-board promotion  
  - Multi-layer promotion
