# MATRIX MODE (8×8×8 / 10×10×10) — CONCEPT DOCUMENT

## Overview
Matrix Mode introduces a full volumetric 3D space. Instead of stacked boards or cube surfaces, the board becomes a true 3D cube of playable cells.

- **8×8×8:** 512 cells  
- **10×10×10:** 1,000 cells  

This is an advanced, experimental mode.

---

## Board Architecture
- Full 3D grid  
- Coordinates: (x, y, z)  
- All cells are playable
