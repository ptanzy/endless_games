# CUBE MODE — CONCEPT DOCUMENT

## Overview
Cube Mode transforms the game board into a cube where each face is playable. Movement wraps across edges, and orientation becomes relative. This is a conceptual mode intended for future expansion.

---

## Board Architecture
- Playable faces of a cube  
- Internal faces only (no outer shell movement)  
- Movement wraps across edges to adjacent faces  

---

## Movement Rules
- Orientation is relative  
- No gravity  
- Movement is continuous across cube surfaces  
- Pawn direction and promotion zones not yet defined  

---

## Starting Positions
Options include:
- Opposite faces  
- Adjacent faces  
- Corner-based deployment
