# CHESS — GAME OVERVIEW

## Purpose
This document provides a high‑level overview of Chess within the EndlessGames ecosystem. It explains what Chess is on this platform, how it fits the EndlessGames identity, and how players experience it across modes, rules, UI, and progression.

---

## What Chess Is on EndlessGames
Chess on EndlessGames is a modular, extensible, multi‑topology strategy game built on the EndlessGames platform principles:

- **Zero moderation** (preset‑only communication)
- **Player‑first monetization** (cosmetic‑only, no power)
- **Unified identity** (shared cosmetics, ranks, achievements)
- **Creator‑friendly** (Tube, TV, Moments integration)
- **Safe for all ages** (gameplay‑only media, no external audio)

Chess is designed to be:
- Competitive  
- Replayable  
- Modular  
- Variant‑friendly  
- Creator‑friendly  
- Platform‑integrated  

---

## Supported Modes
Chess supports multiple board architectures:

- **Single Board** (classic)
- **Double Board**
- **Triple Board**
- **Cube Mode**
- **Matrix Mode**
- **Fan‑Fold Mode**
- **Torus Mode**

Each mode is documented in `/modes`.
