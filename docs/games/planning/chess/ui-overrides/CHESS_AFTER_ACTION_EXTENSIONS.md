# CHESS AFTER ACTION SCREEN — UI EXTENSIONS

## Purpose
This document defines chess-specific extensions to the Generic After‑Action Screen. These include chess stats, event timelines, and mode‑specific performance metrics.

---

## Extension Areas

### 1. Chess Stat Modules
Chess injects the following stats:

#### Core Stats
- Moves made
- Pieces captured
- Pieces lost
- Checkmates delivered
- Checks given
- Times in check

#### Value-Based Stats
- Total material captured
- Total material lost
- Net material score

#### 3D Mode Stats
- Vertical transitions
- Layer changes
- Portal uses
- Terrain interactions

---

### 2. Chess Timeline Events
Chess adds events such as:
- Check events
- Checkmate events
- Promotions
- Terrain collapses
- Portal activations
- Layer transitions
- King captures
- Team eliminations

---

### 3. Chess Victory Panel Enhancements
- Highlighted king of the winning player/team
- Mode-specific victory banner (e.g., “Cube Chess Victory!”)

---

### 4. Chess Rewards
Optional:
- XP for captures
- XP for surviving
- XP for checkmates
- XP for 3D movement mastery (Cube/Matrix)

---

## UX Goals
- Provide meaningful post-match insights
- Celebrate strategic play
- Support competitive and casual players alike
